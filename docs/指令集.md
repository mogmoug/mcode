# MCODE的指令集
**一个指令**  
结构：| OPCODE | ARG | ARG1 |  
大小：| 1byte  |1byte|1byte | 总和：3byte  
# 指令集
1. 0x0 退出指令
2. 0x1 赋值加法
3. 0x2 加法
4. 0x3 减法
5. 0x4 乘法
6. 0x5 除法
7. 0x6 条件跳转
8. 0x7 直接跳转
9. 0x8 输出
10. 0x9 输入
11. 0x10 写入内存
12. 0x11 读出内存
13. 0x12 调用函数
14. 0x13 返回

## 退出指令 end
强制结束这个程序  
跳出运行指令的循环（break）

## 条件跳转 izjmp
如果上一次计算结果为0则直接跳转到由arg和arg1组成的地址
